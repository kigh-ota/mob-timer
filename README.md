# Remote Mob Timer

[![CircleCI](https://circleci.com/gh/kigh-ota/remote-mob-timer.svg?style=svg)](https://circleci.com/gh/kigh-ota/remote-mob-timer)

## To run

```sh
npm install && npm run build
npm run start
```
